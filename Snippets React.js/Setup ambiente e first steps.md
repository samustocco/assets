##### Installazione

```shell
cd [nome cartella]

npm -v

node -v

npm create vite@latest [nome progetto]
```

Seguire i comandi del prompt:

```shell
> yes

> React

> TypeScript

> cd [nome cartella]

> npm install
```

Comandi di Vite:

> Dev server
> `npm run dev`

> Preview
> `npm run preview`

> Build
> `npm run build`
